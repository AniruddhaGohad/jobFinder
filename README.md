A Web-App which helps to find potential candidate for your work.

## Requirements

- Node >= 14.17.2
- NPM >= 6.14.13

## Installation

### 1. Front-End

```bash
cd frontend/
npm install
npm run dev

# this will start your project locally
```

### 2. Backend

```bash
cd backend/
npm install
npm start

# this will start your server locally
```
